# AI Smart Fashion Size Predictor

## Live Demo
**[Click here to try the Live App on Hugging Face]([https://huggingface.co/spaces/esraaelhofy1/fashion-size-predictor](https://huggingface.co/spaces/esraaelhofy1/fashion-size-predictor))**

## Description
This project aims to reduce online shopping returns by predicting the perfect clothing size based on user body measurements (height, weight, age) and clothing category.

## Features
- Dynamic clothing style visualization via URL input.
- Real-time size prediction using a trained Scikit-Learn model.
- High-confidence accuracy scores for each prediction.

## Tech Stack
- **Language:** Python
- **ML Library:** Scikit-Learn (Classification)
- **Deployment:** Gradio / Hugging Face Spaces

## Setup Instructions
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`

## Usage Guidelines
- Paste a direct image URL of the product.
- Select your target group and clothing category.
- Enter your physical measurements using the sliders.
- Click 'Predict' to get your size.
